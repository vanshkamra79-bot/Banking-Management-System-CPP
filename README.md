# Banking Management System (C++)

A simple Banking Management System built using C++ and File Handling.

## Features

* Create new bank accounts
* Login using account number
* Deposit money
* Withdraw money
* Check account balance
* Store account data using files

## Concepts Used

* Classes and Objects
* Object-Oriented Programming (OOP)
* Functions
* File Handling

## How to Run

1. Compile the program:

   ```bash
   g++ fbanking.cpp -o bank
   ```

2. Run the program:

   ```bash
   ./bank
   ```
